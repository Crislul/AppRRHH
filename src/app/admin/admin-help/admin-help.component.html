Manual de usuario y ayuda

¿Cómo usar la página?

1. Navega por las diferentes secciones usando el menú lateral izquierdo para dirigirte a la pestaña deseada.

2. Para visualizar el expediente personal dirígete a la opción de expediente del personal para agregar, editar o visualizar tu información.

3. Para generar un permiso de salida o registrar una incidencia personal dirígete al apartado correspondiente en el menú lateral izquierdo.

4. Si necesitas más ayuda, visita el departamento de recursos humanos de la universidad ubicado en el edificio de TI.
