Manual de usuario y ayuda

¿Cómo usar la página?

1. Navega por las diferentes secciones usando el menú lateral izquierdo para dirigirte a la pestaña deseada.

2. Para visualizar el expediente personal dirigite a la opción de expediente del personal para agregar, editar o visualizar tu información.

3. Para generar un permiso de salida o registrar una incidencia personal dirigite al apartado correspondiente en el menu lateral izquierdo.

4. Si necesitas más ayuda, visita el departamento de soporte de la universidad o manda un correo al encargado de sistemas para obtener ayuda.

Para soporte, contáctanos en: sistemas@utrng.edu.mx o rhumanos@utrng.edu.mx
